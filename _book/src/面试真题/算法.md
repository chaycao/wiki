## 1. 乱序算法

```java
public static void shuffle(int[] array) {
    for (int i = array.length-1; i > 0; i--) {
        Random random = new Random();
        int x = random.nextInt(i);
        swap(array, x, i);
    }
}

public static void swap(int[] array, int a, int b) {
    int temp = array[a];
    array[a] = array[b];
    array[b] = temp;
}
```
